# 项目介绍

## 项目概述

通信基建数智化平台是一个集项目管理、设备管理、3D可视化、告警监控于一体的综合性管理平台。平台采用微服务架构，基于Spring Boot + Vue3技术栈构建，支持通信基站的规划、建设、运维全生命周期管理。

## 技术架构

### 后端技术栈
- **框架**：Spring Boot 3.1.10
- **ORM**：MyBatis-Plus 3.5.5
- **数据库**：MySQL 8.0.46
- **安全**：Spring Security + JWT
- **构建工具**：Maven

### 前端技术栈
- **框架**：Vue 3
- **UI库**：Element Plus
- **状态管理**：Pinia
- **路由**：Vue Router 4
- **3D引擎**：CesiumJS
- **构建工具**：Vite 5

## 核心功能

1. **用户认证**：JWT令牌认证，支持登录、登出
2. **项目管理**：项目列表、阶段进度、验收流程
3. **3D可视化**：Cesium地图、基站环境、设备模型
4. **告警管理**：告警列表、处理记录、统计分析
5. **设备管理**：设备信息、维护记录
