# 开发规范

## 一、代码规范

### 1.1 Java代码规范

#### 命名规范

| 类型 | 规范 | 示例 |
|------|------|------|
| 类名 | PascalCase | `UserController` |
| 方法名 | camelCase | `getUserById` |
| 变量名 | camelCase | `userName` |
| 常量 | UPPER_SNAKE | `MAX_COUNT` |
| 包名 | 全小写 | `com.comm.m05` |

#### 方法命名规范

```java
// 查询方法

getXxxPage()           // 分页查询

// 新增方法
saveXxx()              // 保存
addXxx()               // 添加

// 更新方法
updateXxx()            // 更新

// 删除方法

```

#### Controller规范

```java
@RestController
@RequestMapping("/m05/project")
public class ProjectController {
    
    @GetMapping
    public Result<List<Project>> list() {
        return Result.success(projectService.list());
    }
    
    @GetMapping("/{id}")
    public Result<Project> getById(@PathVariable Long id) {
        return Result.success(projectService.getById(id));
    }
    
    @PostMapping
    public Result<Boolean> save(@RequestBody Project project) {
        return Result.success(projectService.save(project));
    }
    
    @PutMapping("/{id}")
    public Result<Boolean> update(@PathVariable Long id, 
                                   @RequestBody Project project) {
        project.setId(id);
        return Result.success(projectService.updateById(project));
    }
    
    @DeleteMapping("/{id}")
    public Result<Boolean> delete(@PathVariable Long id) {
        return Result.success(projectService.removeById(id));
    }
}
```

const hasData = computed(() => dataList.value.length > 0)

// 6. 方法

  loading.value = true
  try {
    // API调用
    dataList.value = []
  } catch (error) {

})
</script>

## 四、数据库规范

### 4.1 表命名规范

- 使用小写字母
- 多单词用下划线分隔
- 带模块前缀

```sql
-- 正确
m03_project
m03_device
m01_user

-- 错误
M05Project
Project
sys_user
```

### 4.2 字段命名规范

| 类型 | 规范 | 示例 |
|------|------|------|
| 主键 | id | id |
| 外键 | xxx_id | user_id |
| 时间 | _time | create_time |
| 状态 | status | status |
| 布尔 | is_xxx | is_deleted |

### 4.3 SQL编写规范

```sql
-- 使用明确字段名，避免SELECT *
SELECT id, username, email, create_time
FROM m01_user
WHERE status = 1
ORDER BY create_time DESC
LIMIT 10;

-- 使用JOIN代替子查询
SELECT p.*, u.username
FROM m03_project p
INNER JOIN m01_user u ON p.creator_id = u.id;
```

### 7.3 XSS防护

```javascript
// 前端输入转义
const escapeHtml = (str) => {
  return str.replace(/[&<>"']/g, (match) => {
    const escapeMap = {
      '&': '&amp;',
      '<': '&lt;',
      '>': '&gt;',
      '"': '&quot;',
      "'": '&#39;'
    }
    return escapeMap[match]
  })
}
```
